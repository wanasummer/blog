vue2中的渲染函数

渲染函数在一些情况下可以简化组件开发，例如，有一个组件

上述代码可以用渲染函数重写,最终的效果一样 渲染函数可以接收createElement属性，createElement属性是一个对象，简写为h,createElement接收三个参数

第一个参数
{String | Object | Function}
一个 HTML 标签名、组件选项对象，或者 resolve 了上述任何一种的一个 async 函数。必填项。

第二个参数
{Object}
一个与模板中 attribute 对应的数据对象。可选。

第三个参数
{String | Array}
子级虚拟节点 (VNodes)，由 createElement() 构建而成， 也可以使用字符串来生成“文本虚拟节点”。可选。

概述:第一个参数一般是HTML标签名(必填)，第二个参数是一个对象为组件的属性，第三个参数是一个数组(最常用)为组件的子节点
